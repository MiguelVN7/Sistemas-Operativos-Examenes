Este es un archivo de prueba para GSEA.

GSEA (Generador de Sufijos Encriptados y Archivados) es una herramienta
que permite comprimir y encriptar archivos usando diferentes algoritmos.

Este archivo contiene:
- Texto simple
- Repeticiones: AAAAAABBBBBBCCCCCC
- Números: 1234567890123456789012345678901234567890
- Símbolos: !@#$%^&*()_+-=[]{}|;':",./<>?

Línea con espacios múltiples:     espacios     aquí     y     aquí

Texto en mayúsculas: ESTE ES UN TEXTO EN MAYÚSCULAS
Texto en minúsculas: este es un texto en minúsculas
TeXtO MiXtO: EsTe Es Un TeXtO eN fOrMaTo MiXtO

Código de ejemplo:
```c
#include <stdio.h>

int main() {
    printf("Hello, GSEA!\n");
    return 0;
}
```
